# unix-linux-basics
A repo to document some basics in Unix/Linux scripting 

## Printing to console

---

Simple printing to the console with ```echo```

In the terminal:
```shell
$ echo Hello World!

>>> Hello World!
```

```shell
$ echo Hello    World!

>>> Hello World! # No extra spaces!
```

```shell
$ echo "Hello   World"'!' # Extra spaces (note the need for '!')

>>>
Hello   World!
```
